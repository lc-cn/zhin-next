## 快速开始

1. 创建新项目：
```bash
pnpm create zhin my-bot
```

2. 进入项目目录：
```bash
cd my-bot
```

3. 安装依赖：
```bash
pnpm install
```

4. 开发模式启动：
```bash
pnpm dev
```
