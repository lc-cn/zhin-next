## 快速开始

```bash
# 创建新项目
pnpm create zhin my-bot

# 进入项目目录
cd my-bot

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 为什么选择Zhin？

### 开发体验优先

- 内置热重载支持
- 完整的TypeScript支持
- 友好的错误提示
- 详细的开发文档

### 灵活的插件系统

- 模块化设计
- 上下文依赖注入
- 中间件系统
- 事件系统

### 多平台支持

- ICQQ协议支持
- OneBot11协议支持
- 易于扩展新平台

### 高性能

- 基于Node.js
- 支持Bun运行时
- 异步处理
- 资源占用低
